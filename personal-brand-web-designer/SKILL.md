# Personal Brand Web Designer

## Overview

This skill transforms your professional identity—CV, projects, portfolio pieces, and purpose statement—into structured web design deliverables including wireframes, site maps, content blocks, and component breakdowns.

**Keywords**: personal website, portfolio, UX/UI design, wireframes, site maps, PRDs, web design, personal branding, landing pages

## When to Use This Skill

Use this skill when you need to:
- Design or redesign a personal website
- Structure your professional portfolio online
- Create wireframes and layouts for personal brand sites
- Develop site maps and information architecture
- Break down web projects into components
- Prepare specifications for web developers or deployment
- Organize CV content into engaging web sections

## Core Capabilities

### 1. Content Structuring

**From your inputs**, I will:
- Analyze your CV, experience, and skills
- Extract key projects and achievements
- Identify your unique value proposition
- Structure your purpose statement into compelling copy

### 2. Information Architecture

I create:
- **Site Maps**: Logical page hierarchies and navigation structures
- **Content Blocks**: Organized sections (Hero, About, Work, Projects, Contact)
- **User Flows**: How visitors navigate through your story
- **Content Hierarchy**: Priority and placement of information

### 3. Layout & Wireframes

I deliver:
- **Page Layouts**: Desktop and mobile wireframe concepts
- **Section Designs**: Hero sections, project showcases, testimonials
- **Component Breakdown**: Reusable UI elements (cards, navigation, CTAs)
- **Responsive Considerations**: Mobile-first design approaches

### 4. Technical Specifications

I provide:
- **PRDs (Product Requirements Documents)**: Clear feature specifications
- **Component Libraries**: Suggested UI components and frameworks
- **Deployment Readiness**: Vercel/GitHub Pages deployment considerations
- **Tech Stack Recommendations**: React, Next.js, Tailwind CSS, etc.

## Output Formats

```

- Headline: [Your compelling headline]

ABOUT SECTION
```

### Component Breakdown

- Hero Component

- Contact Form
- Footer
- Social Links

## Design Process

1. **Discovery**: Gather your CV, projects list, and purpose statement
2. **Content Strategy**: Identify key messages and user goals
3. **Structure**: Create site map and content hierarchy
4. **Wireframing**: Design layouts for each page/section
5. **Component Design**: Break down into reusable components
6. **Specifications**: Provide PRD and technical requirements
7. **Deployment Guide**: Vercel/GitHub deployment checklist

## Best Practices

### Content
- Lead with your strongest work
- Use active, compelling language
- Focus on outcomes and impact
- Make CTAs clear and prominent

### Design
- Maintain visual hierarchy
- Use whitespace effectively
- Ensure mobile responsiveness
- Keep navigation intuitive
- Optimize for fast loading

### Technical
- Use modern frameworks (React, Next.js, Vue)
- Implement SEO best practices
- Ensure accessibility (WCAG standards)
- Optimize images and assets
- Set up analytics

## Getting Started

To use this skill effectively, provide:
1. Your current CV or resume
2. List of key projects with descriptions
3. Your purpose statement or mission
4. Any existing brand guidelines (colors, fonts, style)
5. Target audience description
6. Websites you admire (for style reference)

## Tips for Success

- **Be Specific**: The more context you provide, the better the output
- **Iterate**: Start with structure, then refine layouts
- **Think Mobile-First**: Most visitors will view on mobile
- **Prioritize Content**: Design serves the message, not vice versa
- **Keep It Simple**: Clean, focused designs convert better
