## Document Linking Strategy

```
LINK TYPES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

UPWARD LINKS (MOC - Maps of Content)
Every note should link to a parent MOC

Example:
Note: "Mega Backdoor Roth Strategy"
→ Links to: [[Tax Optimization MOC]]
→ Which links to: [[Finance MOC]]

LATERAL LINKS (Related concepts)
Connect notes on similar topics

Example:
"Real Estate Underwriting" ← → "Capital Allocation Framework"

DOWNWARD LINKS (Supporting details)
From overview to specific examples

Example:
"Product Development MOC"
→ Links to: [[MVP Definition Template]]
→ Links to: [[User Interview Script]]

BI-DIRECTIONAL LINKING:
When you link A → B, also link B → A
```

## Weekly Knowledge Habits

```
WEEKLY KNOWLEDGE MAINTENANCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

MONDAY (5 min): Inbox Zero for Notes
- Process any loose notes from last week
- Tag and file properly

WEDNESDAY (10 min): Link & Connect
- Review 3-5 recent notes
- Add links to relevant MOCs
- Make lateral connections

FRIDAY (15 min): Weekly Review
- What did I learn this week?
- Create 1 new note or update 1 MOC
- Flag anything to review/expand later

MONTHLY (30 min): Deep Cleanup
- Archive completed projects
- Update all MOCs
- Delete obsolete notes
```
